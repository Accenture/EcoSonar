Do not call a function when declaring a for-type loop in order to avoid function calls each iterations. It saves CPU cycles.

Noncompliant Code Example

for ($i = 0; $i <= foo(); $i++) { // NOK
  ......
}

Compliant Solution

$maxI = foo();
for ($i = 0; $i <= $maxI; $i++) {
  .....
}

  OR

for ($i = 0, $maxI = foo(); $i <= $maxI; $i++) {
  .....
}
}
